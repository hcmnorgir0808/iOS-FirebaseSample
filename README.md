# iOS-FirebaseSample

# setup

## 1. CocoaPods
- gemのrubyコマンドでcocoapodsをインストール
```
sudo gem install cocoapods
```

- Podfileに書いたライブラリをインストール
```
pod install
```

## 2. GoogleService-Info.plistを追加
Firebaseのプロジェクト作成時に設定ファイルのダウンロードを行い、
Xcodeのプロジェクトに追加

参考: https://support.google.com/firebase/answer/7015592?hl=ja
